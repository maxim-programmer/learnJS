 Переменная - это "именованное хранилище" для данных.
 Для создания переменной используйте ключевое слово let. Для присвоения 
значения использовать оператор присваивания = :
1 let message; // переменная с названием  "message"
2
3 message = 'Hello' // сохранили строку 'Hello' в переменную "message"
 Объявление нескольких переменных в одной строке:
1 let user = 'John', age = 25, message = 'Hello'
 Объявление нескольких переменных в многострочном стиле:
1 let user = 'John',
2   age = 25,
3   message = 'Hello';
 ...Или даже с запятой в начале строки:
1 let user = 'John'
2   , age = 25
3   , message = 'Hello';
 Ключевое слово var – почти то же самое, что и let. Оно объявляет 
переменную, но немного по-другому, «устаревшим» способом.
 Повторное объявление переменной вызывает ошибку.
 Два ограничения, касающиеся имён переменных:
1. Имя переменной должно содержать только буквы, цифры или символы $ и _ .
2. Первый символ не должен быть цифрой.
 Если имя содержит несколько слов, обычно используется верблюжья нотация, 
то есть, слова следуют одно за другим, где каждое следующее слово 
начинается с заглавной буквы: myVeryLongName.
 Регистр в названии переменной имеет значение.
 Нелатинские буквы разрешены, но не рекомендуются.
 Существует список зарезервированных слов, которые нельзя использовать 
в качестве имён переменных, потому что они используются самим языком.
 Чтобы объявить константную, то есть, неизменяемую переменную, используйте 
const вместо let:
1 const myBirthday = '18.04.1982'
 Константу нельзя перезаписать!
 Константы с именами, записанными заглавными буквами, - это значения, 
известные до выполнения. Константы, записанные как обычно, - это значения, 
которые вычисляются во время выполнения сценария.
 Переменные должны быть названы таким образом, чтобы мы могли легко понять, 
что у них внутри.
